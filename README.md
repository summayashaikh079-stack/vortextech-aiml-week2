# vortextech-aiml-week2
Week 2 - Classification Model on Titanic Dataset using Logistic Regression | Vortex Tech AI &amp; ML Internship 2026

## What I Did
- Used the cleaned Titanic dataset from Week 1
- Selected features: Pclass, Sex, Age, Fare, SibSp, Parch
- Converted categorical column (Sex) into numeric using pd.get_dummies()
- Split data into training (80%) and testing (20%) sets
- Trained a Logistic Regression model using scikit-learn
- Evaluated the model using Accuracy, Precision, Recall, and F1 Score
- Wrote a summary interpreting the results and ideas for improvement

## Results
- Accuracy: 81%
- Precision: 80%
- Recall: 71.6%
- F1 Score: 75.7%
